# Quality Gate Standard Corporate - SonarQube 2026

Owner : Equipe Plateforme Qualite

## Quality Gate : Corporate Quality Gate 2026

Conditions sur le NOUVEAU CODE :
- Bugs Blocker et Critical : 0
- Vulnerabilites critiques et majeures : 0
- Hotspots securite : 100 pourcent revus
- Couverture de code : superieure ou egale a 80 pourcent
- Lignes dupliquees : inferieure a 3 pourcent
- Maintainability Rating : A
- Reliability Rating : A
- Security Rating : A

Conditions sur le CODE GLOBAL (non bloquant, tendance) :
- Security Rating : superieure ou egale a B
- Reliability Rating : superieure ou egale a B

## Comportement

PASSED : toutes conditions remplies, release autorisee
FAILED : au moins une condition non remplie, release bloquee

## Derogation

Exceptionnelle et limitee a 1 sprint.
Accord du Tech Lead et du PO requis.
Tracee dans le registre des derogations.

## Ecocode 2026

Regles ecocode actives en mode Warning (non bloquant).
Objectif 2027 : passage en mode bloquant sur regles critiques ecocode.

## Evolution

Toute modification passe par le Comite Qualite mensuel.
